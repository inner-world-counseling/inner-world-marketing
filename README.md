# Inner World Counseling Services

Welcome to the official web repository for **Inner World Counseling Services**. This project houses the secure, high-performance marketing platform for our telehealth practice.

## About the Practice
Inner World Counseling Services provides professional, trauma-informed telehealth and online therapy. We specialize in helping individuals navigate anxiety, depression, and personal wellness through modern clinical approaches.

## Tech Stack & Security
To ensure a secure and fast experience for our clients, this site is built with:
* **Hosting:** Netlify (Global Edge Network)
* **Security:** Cloudflare (WAF & DDoS Protection)
* **Integration:** Secure SimplePractice clinical widgets
* **Version Control:** GitHub (Transparency & Auditability)
